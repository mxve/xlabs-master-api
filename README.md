# xlabs-api

[xlabs.plutools.pw/api/](https://xlabs.plutools.pw/api/servers)

## Endpoints
All endpoints are json, unless specified otherwise.

- ```/servers```
  - All servers
- ```/servers/<game>```, game = iw4x, iw6x, s1x
  - Servers of specified game
